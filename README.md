Este repositorio contiene todos los ejercicios realizados durante el curso
de Introducción a JavaScript dictado por ADA ITW.

- Funciones - Arrays y objetos

### Objetivo

Demostrar mi capacidad para aplicar los conocimientos adquiridos en el curso, resolver ejercicios prácticos y trabajar con herramientas de desarrollo
modernas.
